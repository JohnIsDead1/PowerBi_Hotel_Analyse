🏨 Power BI Hotel Revenue Analysis
📊 Project Overview

This project presents an interactive dashboard built in Power BI to analyze hotel performance across multiple dimensions such as revenue, bookings, discounts, and parking demand.

The dashboard helps uncover trends, compare hotel types, and support data-driven decision-making in the hospitality domain.
